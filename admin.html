How to add a new project
  1. Copy your project photo into the images/ folder.
  2. Fill in the form — title, category, image path (e.g. images/photo.jpg), and a description.
  3. Click Add Project or press Ctrl + Enter.
  4. Click Save projects.json. If that fails, click Download JSON and replace the file in your website folder.
  5. Refresh the portfolio page — your project will appear automatically.
