📚 How to add a new project (no coding needed)

  1. Copy your project photo into the images/ folder inside the website folder.
  2. Fill in the form on the right — title, category, image filename (e.g. images/myproject.jpg), and a short description.
  3. Click Add Project.
  4. Click Save projects.json (Chrome/Edge will save it directly). If that doesn't work, click Download JSON and replace the projects.json file in your website folder with the downloaded file.
  5. Refresh the portfolio page — your project will appear automatically.
